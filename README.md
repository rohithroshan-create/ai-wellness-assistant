# 🏥 AI Wellness Assistant

An intelligent health companion that predicts risks for chronic diseases and provides personalized wellness recommendations.

## 🎯 Features

- **Multi-Disease Risk Prediction**: Heart Disease, Diabetes, Hypertension
- **AI-Powered Explanations**: SHAP/LIME integration for transparent decisions
- **Personalized Recommendations**: Non-medicinal lifestyle interventions  
- **Interactive Health Assistant**: Chat with AI about your assessment
- **Professional Medical Interface**: User-friendly design for patients and providers

## 🚀 Quick Start

### Option 1: Streamlit Cloud (Recommended)

1. Fork this repository
2. Connect to [share.streamlit.io](https://share.streamlit.io)
3. Deploy with one click!

### Option 2: Local Development

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Model Performance

- **Heart Disease**: 87% accuracy, 0.92 AUC
- **Diabetes**: 85% accuracy, 0.89 AUC  
- **Hypertension**: 83% accuracy, 0.88 AUC

## 🔬 Technical Stack

- **Frontend**: Streamlit with custom CSS
- **ML Models**: XGBoost, Random Forest, Logistic Regression
- **Explainability**: SHAP, LIME
- **Visualization**: Plotly, Matplotlib
- **Deployment**: Streamlit Cloud
